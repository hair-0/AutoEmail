# 快速开始指南

## 🚀 5分钟快速上手

### 第一步：准备邮箱（2分钟）

1. 登录网易163邮箱：https://mail.163.com
2. 点击"设置" → "POP3/SMTP/IMAP"
3. 开启"POP3/SMTP服务"
4. 点击"授权密码管理" → 发送短信验证
5. 获得16位授权码（**重要**：不是登录密码）

### 第二步：启动程序（1分钟）

**方法A：使用exe文件（推荐）**
```
双击 邮件发送工具.exe 即可
```

**方法B：使用Python源码**
```bash
cd AutoEmail
pip install -r requirements.txt
python main.py
```

### 第三步：配置邮箱（1分钟）

1. 打开程序
2. 在"邮箱设置"中填写：
   - 发件邮箱：yourname@163.com
   - 授权码：刚才获取的16位授权码
3. 点击"保存配置"

### 第四步：添加联系人（1分钟）

**方法A：手动添加**
1. 切换到"通讯录"标签页
2. 填写姓名、邮箱（职称和院系可选）
3. 点击"添加"

**方法B：批量导入（推荐）**
1. 准备CSV文件（可参考"联系人导入示例.csv"）
2. 点击"批量导入CSV"
3. 选择CSV文件

### 第五步：发送邮件（1分钟）

有两种发送方式：

**方式一：立即发送（不可撤回）**
1. 切换到"发送邮件"标签页
2. 填写邮件主题和内容（可使用占位符：`{姓名}`, `{职称}`, `{院系}`）
3. 点击"从通讯录添加"选择收件人
4. （可选）点击"预览邮件"查看效果
5. 点击"立即发送"

**方式二：加入发送队列（可撤回）** ✨推荐
1. 填写邮件内容并选择收件人
2. 点击"加入发送队列（可撤回）"
3. 邮件会在30秒后自动发送，期间可以撤回
4. 切换到"发送队列（可撤回）"标签页查看和管理

## 💡 使用技巧

### 技巧1：保存常用模板
在"邮件模板"标签页中，可以保存多个模板，方便不同比赛使用。

### 技巧2：添加附件
点击"添加附件"可以添加比赛文件、日程表等附件。

### 技巧3：预览功能
发送前务必点击"预览邮件"，检查内容是否正确。

### 技巧4：查看历史
在"历史记录"标签页可以查看所有发送记录，双击可查看详情。

### 技巧5：批量导入
使用Excel编辑CSV文件，可以快速添加大量联系人。

### 技巧6：撤回邮件 ✨新增
使用"加入发送队列（可撤回）"方式发送邮件，在30秒倒计时期间可以撤回，防止误发。

## ⚠️ 常见问题速查

| 问题 | 解决方案 |
|------|---------|
| 登录失败 | 检查是否使用授权码而非登录密码 |
| 邮件进垃圾箱 | 首次发送正常，后续会自动恢复正常 |
| 附件发送失败 | 检查文件大小是否超过20MB |
| 程序无法启动 | 确认Python版本3.7+ |
| CSV导入失败 | 确保包含"姓名"和"邮箱"列 |

## 🎯 核心功能一览

| 功能 | 位置 | 说明 |
|------|------|------|
| 发送邮件 | "发送邮件"标签页 | 主功能，支持模板和附件 |
| 发送队列管理 | "发送队列（可撤回）"标签页 | 查看待发送邮件，支持撤回 |
| 管理模板 | "邮件模板"标签页 | 保存、编辑、删除模板 |
| 通讯录管理 | "通讯录"标签页 | 添加、编辑、导入、导出联系人 |
| 查看历史 | "历史记录"标签页 | 查看所有发送记录 |
